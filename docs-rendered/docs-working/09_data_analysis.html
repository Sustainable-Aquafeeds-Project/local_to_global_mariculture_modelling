1 Overview

This document does some additional data analysis on the outputs before presenting numbers needed for the manuscript. In particular, we fit a series of mixed-effects models to determine how much of the variance can be attributed to fixed effects (different feeds), random effects (different farms) and residual effects (Monte-Carlo sampled individual fish).

    ) %>% as.factor()
  )

For total nitrogen and carbon, we use the total value per biomass rather than total value. This is because total value is highly dependent on farm population, which is implied rather than measured and gives far too much weight to the random effect of farm location.

Successively removing random effects shows that farm_ID is the only one that really matters wihtout overcomplicating the model structure. In the final model:

  • Feed and nested feed_level effects are fixed effects
  • Farm_ID is controlled as a random intercept
  • The residual represents individual fish variation
Fit actual model for total_nitrogen 
VP_nitrogen <- file.path(data_analysis_path, "VP_modelfinal_total_nitrogen.qs")
if (!file.exists(VP_nitrogen)) {
  VP_nitrogen <- lmer(
    value_perbiom ~ feed/feed_level + (1 | farm_ID), 
    data = fish_ends_nitrogen, REML = TRUE,
    control = lmerControl(optimizer = "bobyqa",optCtrl = list(maxfun = 2e5))
  )
  qsave(VP_nitrogen, file.path(data_analysis_path, "VP_modelfinal_total_nitrogen.qs"))
} else {
  VP_nitrogen <- qread(VP_nitrogen)
}

3 Shifts in latitude

From Julia: Can we say With mean digestibolities there is X% more carbon/nitrogen loading per ton of fish produced per degree (either lat or temperature) for the reference MD feed. Shifts towards PD result in an equal/different? X % more C across all regions? with only X% more /less for PD (all based on mean digestibilty). Then N results follow slightly different pattern with AD performing least best. (& MD best in terms of lowest N loadings). Then state whether the differences vary/d not vary across space.

Explore percentage change in carbon loading per biomass relative to MD feed 
farm_ends_carbon %>%
  mutate(mean_value_perbiom = mean_value_perbiom %>% set_units("g g_fish-1") %>% set_units("kg t_fish-1")) %>% 
  filter(feed_level == "mean") %>% 
  select(-mean_value, -sd_value, -sd_per_biom, -n) %>% 
  pivot_wider(names_from = feed, values_from = mean_value_perbiom) %>% 
  mutate(
    PD_change = (marine_dominant_biomar-plant_dominant_biomar)/marine_dominant_biomar,
    AD_change = (marine_dominant_biomar-animal_dominant_biomar)/marine_dominant_biomar
  ) %>% 
  select(-contains("biomar")) %>% 
  pivot_longer(cols = contains("change"), names_to = "feed", values_to = "perc_change") %>% 
  ggplot(aes(x = farm_ID, y = perc_change, colour = feed)) + geom_point()
